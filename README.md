# Minizinc_Exercises
Some Constraint Satisfaction Problem (CSP) exercises solved with Minizinc.

## Execution ⚙️

Download every .mzn exercise and execute it using Minizinc.
To execute an exercise just open it using Minizinc and press "Run" button.
